# Reaction-Wheel-Attitude-Control-System-in-CubeSats

May 2024 - Jul 2024

Summary: Designed single-axis reaction wheel control system, to be later extended to 4-axes

Application: Rapid recovery of attitude from disorientation during payload recovery

Hardware Used: Arduino Uno R3, Polulu DC Motor with Encoder, Motor Driver, IMU Sensor, Bluetooth Module, Relay, LiFe Battery

Test Apparatus Used: FloatBed, Air Compressor (to power FloatBed)
